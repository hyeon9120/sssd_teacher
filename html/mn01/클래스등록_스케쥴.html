◀ 스케줄은 어떻게 등록하나요?
  1. ① 기간 선택
    • 1번의 강의로 끝난 다면 원데이 추천!
    • 긴 기간의 강의가 필요하다면 장기 추천!
  2. ② 소요시간
    • 시작시간으로부터 클래스가 진행될 시간 입니다.
    • 최소 1시간 ~ 최대 8시간 까지 가능합니다.
    • 8시간의 경우 `종일` 강의로 인식됩니다. 추가 스케줄 등록 시 참고해주세요.
    • 수강생이 있을 경우 수정/삭제가 불가하니 유의해서 선택 해주세요.
  3. ③ 날짜 선택
    • 클래스가 이루워질 날짜와 시간을 선택해주세요.
    • 날짜 등록은 ‘오늘’ 기준으로 최소 5일 뒤부터 생성 가능합니다.
    • 수강생이 있을 경우 해당 스케줄은 수정/삭제가 불가하니 유의해서 선택 해주세요.
◀ 모집인원
  • 최소인원과 최대인원에 따라 클래스가 운영됩니다.
  • 최소인원이 미달될 경우 클래스 시작 2일전 오후 5시에 자동취소되며, SMS로 강사/수강생에게 안내 드립니다. (전액 환불)
◀ 가격설정
  • 재료비 포함여부, 또는 별도인지에 대해 수강생이 신청 전 먼저 확인해 볼 수 있습니다.
  • 선택항목에 따라 다른경우 등록한 항목을 수강생이 청구할 경우 해당 가격으로 부과되게 됩니다.
◀ 상세정보등록 (타이틀)
  • 상세정보 등록의 경우 수강생에게 해당 클래스를 좀 더 매력적으로 알릴 수 있는 요소가 됩니다.
  • 이런점이 좋아요 / 이런분들에게 추천해요 두 요소 필수가 아닌 선택항목이기 때문에 미 등록히 해당 영역이 미노출됩니다.
◀ 클래스 완성작
  • 클래스 완성작의 경우 수강생에게 해당 클래스 수강시 만들 수 있는 작품의 예시를 미리 선보여 클래스를 좀 더 매력적으로 알릴 수 있는 요소가 됩니다.
  • 필수가 아닌 선택항목이기 때문에 미 등록히 해당 영역이 미노출됩니다.
◀ 제공사항
  • 장소에 연결된 제공항목을 선택해 주세요. (필수)
  • 제공하는 기본 항목을 선택하면 수강생들이 좀 더 편리하게 이용할 수 있습니다.
◀ 추가 제공사항
  • 기본 제공사항이 없으신가요? 추가 제공사항에 자세히 적어주세요.
  • 최대 5개까지 작성 가능합니다.
◀ 태그
  • 태그를 추가해 클래스 검색에 적극 활용하세요.
◀ 추천쿠폰(배너) 연결
  • 솜씨당에서 제공하거나 강사가 직접 발행한 쿠폰을 연결해 보세요.
  • 현재 등록 가능한 쿠폰 순으로 노출되며, 솜씨당이 발행한 쿠폰은 추후 클래스 수강료 정산시 혜택정산으로 따로 정산받으실 수 있습니다.
  • 쿠폰 연결로 클래스 수강생 유치에 적극 활용하세요.
◀ 작품갤러리 사진
  • 강사의 다른 작품사진을 게시할 수 있는 작품갤러리 사진은 앱 / 웹 내 디테일 화면 하단에 노출되기 때문에 고화질의 가로 사진을 권장해 드립니다.
  • 수강생들에게 공개되는 정보로 강사님의 솜씨를 수강생이 확인하고 클래스 신청에 영향을 줄 수 있기 때문에 퀄리티 높은 사진을 등록해 수강생 유치에 적극 활용해보세요.
