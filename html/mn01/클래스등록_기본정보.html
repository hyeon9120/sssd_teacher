◀ 클래스 구분이란 뭔가요?
  1. ① 일반 클래스
    • 작가가 등록하는 일반적인 클래스로 모든 작가가 등록 할 수 있습니다.
  2. ② 작가매칭 클래스
    • 일반 유저가 올린 함께해요 게시글에 지원하여 해당 함께해요에 매칭된 클래스로 구분되어 노출되는 클래스 입니다.
    • 또한, 클래스 정보 등록 시 해당 함께해요에 신청한 유저에게 push 알림이 발송되어 유저가 클래스 정보 확인 후 클래스 신청을 확정하여 진행할 수 있습니다.
◀ 클래스 명과 소개는 어떻게 써야 할까요?
  1. ① 클래스 명
    • 클래스를 대표하는 명칭이며, 수정이 불가하니 유의해서 작성해주세요.
    • 명칭이 길 경우 말줄임 … 됩니다.
  2. ② 클래스 소개
    • 클래스에 대한 간단한 소개 글을 작성해주세요.
    • 자세한 내용은 "커리큘럼" 에 적어주세요.
◀ 카테고리는 어떻게 선택 하나요?
  • 해당 클래스를 검색 및 분류 시에 사용됩니다.
  • 1차 카테고리를 선택하시면, 2차 선택을 하실 수 있으며, 중복 선택이 불가합니다.
  • 카테고리는 수정이 불가하니 유의해서 선택해주세요.
◀ 위치정보
  • 위치정보를 정확히 입력해 주셔야 추천지역 및 클래스 정보 화면에 정확한 위치가 안내됩니다.
  • 등록하신 위치정보 기반으로 솜씨당의 시스템에서 추천지역을 설정하여 클래스 검색 시 노출되게 되니 정확한 위치 정보를 입력해 주셔야 합니다.
◀ 대표사진
  • 대표사진은 앱 / 웹 내 클래스 리스트 및 디테일 화면에서 노출되기 때문에 고화질의 가로 사진을 권장해 드립니다
  • 최대 10장까지 등록하실 수 있으며, 첫번째로 위치한 사진이 대표사진으로 자동 지정되어 클래스 리스트/ 디테일 화면에서 가장 먼저 노출되게 됩니다.
◀ 임시저장이 뭔가요
  • 작성 도중 임시저장을 하시면, 웹이 오류로 강제로 닫히거나 재접속 시 저장한 시점부터 데이터를 확인하실 수 있습니다.
